# Github

(opens new window)

Puedes descargar la última versión de Chart.js en GitHub (opens new window).

Si descargas o clonas el repositorio, debes build Chart.js para generar los archivos dist. Chart.js ya no viene con versiones de lanzamiento precompiladas, por lo que se recomienda enfáticamente una opción alternativa a la descarga del repositorio.
